The ddi package

Portability interface with the Decision Diagram package.

By Gianpiero Cabodi and Stefano Quer

The DDI package provides functions to manipulate the following objects:
  • boolean functions (Ddi_Bdd_t)
  • arrays of boolean functions (Ddi_Bddarray_t)
  • variables (Ddi_Var_t)
  • arrays of variables (Ddi_Vararray_t)
  • sets of variables (Ddi_Varset_t).
It can be viewed both as a portability interface, concentrating all dependancies from the BDD package, and as an upper level, manipulating the previously listed BDD based data types. A particular feature, distinguishing this interface from other ones, is the support for partitioned (conjunctive as well as disjunctive) functions. This implementation is written on top of the CUDD package.
