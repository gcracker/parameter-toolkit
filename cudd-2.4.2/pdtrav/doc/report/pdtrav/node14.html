Setting Parameters

All packages require parameter setting for their operations. For each package, proper functions support set/show and restoring of default settings package provides several functions to set the parameters that control various functions. The value of these settings are usually stored into the relative manager structure. A lot of commands can receive additional options. These options usually overwrite the generic settings.
