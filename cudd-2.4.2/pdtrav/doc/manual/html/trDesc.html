The tr package

Funtions to manipolate Transition Relation

By Gianpiero Cabodi and Stefano Quer

The Tr (Transition Relation) package provides a set of functions to manipolate the Transition Relation.
The transition relation can be built using the monolithic form or clustering heuristics.
Squaring it, transitive closure, is also possible.
External procedures included in this package:
  • Tr_BddarrayAndExist()
  • Tr_CreateTrv()
  • Tr_BddarraySort()
