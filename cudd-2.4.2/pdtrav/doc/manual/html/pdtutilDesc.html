The pdtutil package

Utility functions for the PdTRAV package.

By Gianpiero Cabodi and Stefano Quer

This package contains a set of utility functions shared in the PdTRAV package.
In this package are contained functions to allocate and free memory, open and close files, deal with strings, read and write variable orders, etc.
